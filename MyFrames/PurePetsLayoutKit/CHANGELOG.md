# Changelog

## 1.0.0

- Rebuilt collection layout management in Swift and SwiftUI.
- Added Objective-C runtime-compatible bridge classes.
- Added per-item ratio sizing and async invalidation.
- Added adaptive grid, full-width list, horizontal-row, carousel, and masonry modes.
- Added loading, empty, error, accessibility, RTL, keyboard, and motion-aware UI primitives.
- Added deterministic geometry and regression tests.
